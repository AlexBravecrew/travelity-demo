# Visual references

These HTML files are read-only design references. They are NOT part of the
marketing site build — they exist so we can pull animation timings, exact
spacing values, SVG paths, and color rotations when porting designs into Astro
components.

## Files

- **`travelity-home-v2_4.html`** — Canonical home-page reference. Source of
  truth for Phase 3 (Hero, Channels, Features, Parallax, Pricing, GoLive,
  ClosingCTA). Final cool-palette version with all 6 sections.

- **`index.html`** — Multi-page demo from before the home-page redesign.
  Reference for Phases 5–7 (Solutions cluster, Audience cluster, Book Demo).
  Pull _page structure and section anatomy_ from here, but apply the _visual
  language_ from `travelity-home-v2_4.html` (newer tokens, italic-em pattern).

## How to use

Open these files directly in a browser. Don't import them, don't bundle them,
don't reference them from the Astro build. They are documentation, not code.
